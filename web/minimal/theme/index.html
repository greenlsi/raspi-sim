SOBRE EL PROYECTO

ARM HARDWARE EMULATOR se realizó en el marco de un proyecto de investigación como base de un TFM, y posteriormente ampliado para un TFG, dentro del grupo GreenLSI. Nació con el objetivo diseñar una herramienta de utilidad a fines didácticos. Actualmente el emulador es funcional pero es solo un inicio, se sigue mejorando y agregando funcionalidades. La meta u objetivo final, es conseguir un simulador de plataformas ARM modular que incluya la simulación de sensores y actuadores externos de todo tipo y que permita llevar a cabo todas las funcionalidades que se conseguirían con el dispositivo real.

Green LSI

EMULADOR

Hoy en día, existe una variada oferta de plataformas basadas en ARM que abarcan una buena parte del mercado y que gozan de una amplia comunidad. RaspberryPi, Discovery de ST o BeagleBone, son algunas de ellas. La plataforma que se empleó como base del proyecto fue RaspberryPi B+. El modelo inicial incluye un System-on-a-chip Broadcom BCM2835, que contiene un procesador central(CPU) ARM1176JZF-S a 700 MHz

La emulación se realizó usando QEMU, se crearon los dispositivos periféricos, los drivers respectivos, y se implementó un mecanismo de comunicación basado en websocket. La interfaz fue programada en HTML5 y js.
